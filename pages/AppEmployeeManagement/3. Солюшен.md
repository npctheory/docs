##### VS Code
`Ctrl+Shift+P` Generate Assets for Build and Debug  

##### CLI: Создание солюшена и проекта
```bash
dotnet new sln --name DemoAppMvc
dotnet new web -o Mvc
dotnet sln add ./Mvc
dotnet build
```

##### Запуск проекта
```
dotnet build
cd ./Mvc
dotnet run
```
