### Новый репозиторий
```bash
git init  
git add .  
git commit -m "init"  
git remote add origin https://github.com/aq-coding/github-actions.git  
git push -u origin master
```
### Новая ветка
```
git checkout -b dev
git add .  
git commit -m "init"  
git push origin dev
```
